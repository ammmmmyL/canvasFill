Get Started

Fill Up empty blocks on your website with a few lines of code. No more time needed to spend on the tedious process of adjucting the positions and values of various Html and CSS elements. Start using the library today by simply include the following file. The returned elements will be styles series of node elements that can be directly added as a child node. You can also add additional styles to individual elements.

   < !-- JS Library script -->
<script type="text/javascript" src='./js/CanvasFill.js'> <script />
API Specifications
Object Declaration
const splitImage = {
     CreateSplitImage,
     fontStyle: "Helvetica",
     fontColor: "black",
     textBackground: "#ffffff00"
}
splitImage.CreateSplitImage(img, caption, width, height, left, right)
Property Default Type Description
img null image The image you want to style
caption null string The caption you want to style
width null number The width of the canvas you want to style in px
height null number The height of the canvas you want to style in px
left null number The left split size the image in decimal [0 - 1]
right null number The right split size the image in decimal [0 - 1]
fontStyle Helvetica string Font family style you want to apply to the caption
fontColor black string Font color you want to apply to the caption
textBackground #ffffff00 string Background color you want to apply to the caption, default is transparent
